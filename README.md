# 🇮🇳 VIndia – Vehicles of India

🚗 An online platform to explore different categories of Indian vehicles with images, prices, and filtering options.

🔗 **Live Site:** [vehicles-of-india-vindia.netlify.app](https://vehicles-of-india-vindia.netlify.app)

---

## 📌 Project Overview

**VIndia** is a web-based vehicle showcase platform developed using HTML, CSS, Bootstrap, and JavaScript.  
It allows users to browse different vehicle categories like bikes, cars, e-vehicles, and more — inspired by real-world UI experiences.

This project was built to strengthen frontend development skills and deploy a working UI prototype for an automotive gallery.

---

## 🧩 Features

- ✅ Clean and responsive UI  
- ✅ Category-based filtering (e.g. Bikes, Cars, EVs)  
- ✅ Hover effects and image transitions  
- ✅ Bootstrap-powered design  
- ✅ Fully responsive layout  
- ✅ Deployed using **Netlify**

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript  
- **Deployment:** Netlify
